## Requirements

Make sure you have the following software and libraries installed:

- Python 3.9.12
- pandas 
- scikit-learn 1.1.3
- xgboost
- numpy

You can install the required libraries using the following command:

```bash
pip install pandas scikit-learn xgboost numpy

```
